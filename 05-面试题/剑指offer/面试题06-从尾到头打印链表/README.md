#### 面试题06. 从尾到头打印链表

> 难度：简单

输入一个链表的头节点，从尾到头反过来返回每个节点的值（用数组返回）。

**示例:**

~~~go
/**
 * Definition for singly-linked list.
 * type ListNode struct {
 *     Val int
 *     Next *ListNode
 * }
 */
func reversePrint(head *ListNode) []int {
}

输入：head = [1,3,2]
输出：[2,3,1]
~~~



**限制：**

```
0 <= 链表长度 <= 10000
```
